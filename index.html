emscripten-worker-leak-demo

The startup buttons below do as follows:

  • Load Script: Loads the Emscripten-generated JavaScript script file
  • Bootstrap Module: Runs the Emscripten-generated module bootstrapping function DemoModule()
  • Initialize Heavy Object: Creates a large object. That's it.
  • Run: Queries the size of the large object, and prints the result to the console.

The shutdown buttons do as follows:

  • Delete Heavy Object: Destroys the large, heavy object. That's it.
  • Destroy Module Reference: Removes the reference to the Emscripten-generated module.
  • Unload Script: Removes the Emscripten-generated JavaScript script file from the document. This isn't really important or impactful, but I included it anyways.
